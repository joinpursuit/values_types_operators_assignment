## Problem Three

Create a truth table for the expression A || B.

For reference, here is a truth table for the expression A && B:

|   A   |   B   | A && B | 
|-------|-------|--------|
| true  | true  | true  |
| false | true  | false |
| true  | false | false |
| false | false | false | 

## Problem Four

For reference, here is a truth table for the expression A && !B:

|   A   |   B   |   !B   | A && B | 
|-------|-------|--------|--------|
| true  | true  | false  | false  |
| false | true  | false  | false  |
| true  | false | true   | true   |
| false | false |  true  | false  |
